# Ops: Deploy / Run Records

This folder stores short records of completed deploys and operational verification runs.

## Naming
- `YYYY-MM-DD.md`

Example:
- `2026-04-19.md`

## What to include
- A brief result summary (✅/⚠️)
- What was verified (env/secrets, Supabase RLS/migrations, email send, E2E)
- Where audit exports were uploaded (shared drive path)
